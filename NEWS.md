landscapeR v1.1 (Release date: 2016-08-30)

* Updated and expanded vignette 
* Added copyright notes in COPYRIGHT file
* Fixed author list to CRAN standards

NEW:

* Added NEWS.md file
* Added start up messages with copyright notice
